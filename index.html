Project Documentation: Escape The Mazerooms

Introduction
"Escape The Mazerooms" is an interactive 2D horror game that takes players through a series of mazes and tasks with the goal of achieving liberation. The game is set in an atmospheric environment, where each level brings new challenges and surprises.

Main Software Parts / Modules and Their Interconnection
The project is divided into several Python scripts that serve different aspects of the game:

  • EscapeTheMazerooms.exe: Launches the entire game.
  • main.py: Initializes the game and manages transitions between various modules.
  • level0.py to level3_end.py: Responsible for the logic of individual game levels.
  • menu.py: Displays the main menu and ensures navigation.
  • intro.py and outro.py: Scripts for playing intro and outro videos.
  • settings.py: Stores game configuration.
  • assets/: Contains all data (images, music, etc.)

Description of Main Functions, Service Description
The game provides the following functionalities:

  • A game loop with levels of increasing difficulty.
  • Video playback using the MoviePy library.
  • User settings that can be adjusted in settings.py.

Architecture Sketch
The game's architecture is based on a client-server model, where the client is the user interface and the server is the game logic, including state management and game saving. The backend and frontend communicate through defined Python classes and functions.

Environment
The game was developed and tested in the Python 3.8 environment with the Pygame library and is designed to run on the Windows platform.

Source Code Location
The game's source code is available on GitHub, allowing for easy version management and community access.
