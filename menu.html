menu module

class menu.MainMenu[source]

Bases: object

A class that creates and manages the main menu for a game.

The main menu allows the player to choose from playing the game, viewing the tutorial, accessing the magazine, or quitting the game. It handles user interactions and transitions between different states based on user input.
